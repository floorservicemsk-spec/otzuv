БЫЛО (v1.0):                         СТАЛО (v2.0):
────────────────────────────────────────────────────────────────────────────────
Поддомены (subdomain)             →  Уникальные URL (form_id)
client.yourservice.com            →  yourservice.com/form/abc123

Фиксированные поля                →  Конструктор полей
6 жёстких полей                   →  Любые кастомные поля

Данные в колонках                 →  JSON формат
phone, name, email...             →  form_data (longtext JSON)

Wildcard DNS нужен                →  Обычный домен достаточно
*.yourservice.com                 →  yourservice.com

Нельзя изменить поля              →  Drag & Drop редактор
Только дизайн                     →  Поля + дизайн

Подробнее → REFACTORING_V2.md
